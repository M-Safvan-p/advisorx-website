# AdvisorX Website

## Overview

This is the official website developed for the AdvisorX team. The project was built as a responsive frontend to represent the AdvisorX brand online and communicate its services through a modern single-page web experience.

## Client

AdvisorX is a venture ecosystem platform focused on startup incubation, investor access, and founder community building. It positions itself as a structured platform that supports early-stage founders, connects validated startups with investors, and builds a governed community for founders, operators, and aspiring entrepreneurs.

## Project Scope

The website was developed to present AdvisorX's core offerings across:

- Venture Incubation
- Investor Access
- Community

The focus of this project was to deliver a clean, responsive, and brand-aligned frontend experience suitable for public deployment.

## Stack

- React
- Vite
- Tailwind CSS
- React Router DOM
- Lucide React

## Deployment

This project uses `BrowserRouter`, so single-page application route fallback is required in production. The repository includes `public/_redirects` for Netlify-style deployments. Equivalent route rewrite rules should be configured when deploying to other platforms.
